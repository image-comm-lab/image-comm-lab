  • A Mature-Tomato Detection Algorithm Using Machine Learning and Color Analysis

    We proposed a coarse-to-fine algorithm for mature-tomato detection using regular color images. This method can restrain the influence of illumination variation and occlusion. The results on the test images showed a considerable improvement compared with previous methods.

    Guoxu Liu, Shuyi Mao, Jae Ho Kim
